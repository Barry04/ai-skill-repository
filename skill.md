# Project Skills

## Minimal Scope
trigger: scope, overdesign, architecture
content:
* Prefer the smallest in-agent mechanism that solves the task.
* Do not introduce services, APIs, databases, or platform concepts.

---

## Skill Injection
trigger: skill, injection, prompt
content:
* Inject no more than two relevant skills into the prompt.
* Rewrite skill content to fit the current task instead of copying it verbatim.

---

## Skill Compression
trigger: compression, record, learn
content:
* Record a skill only after a solved task reveals reusable experience.
* Keep each skill short, general, and independent of one-off context.

---
